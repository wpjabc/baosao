Why Nemo?

Nemo adds a powerful configuration engine and plugin architecture to the selenium-webdriver wrapper for node.js. It can be paired with most test runners (think mocha, cucumber, etc) and task runners (grunt, gulp) or used as a stand-alone tool for convenience scripts.

Clever Configurability

Using the clever, concise and powerful confit configuration module and its shortstop handler syntax, you have many elegant ways to configure and override.
